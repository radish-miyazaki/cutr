## cutr

```
Rust cut

Usage: cutr [OPTIONS] <--fields <FIELDS>|--bytes <BYTES>|--chars <CHARS>> [FILE]...

Arguments:
  [FILE]...  Input file(s) [default: -]

Options:
  -f, --fields <FIELDS>    Selected fields
  -b, --bytes <BYTES>      Selected bytes
  -c, --chars <CHARS>      Selected characters
  -d, --delim <DELIMITER>  Field delimiter [default: "\t"]
  -h, --help               Print help
  -V, --version            Print version
```
